# 🧮 Calculator Web App

A simple and responsive calculator web application built using **HTML, CSS, and JavaScript**. This project performs basic arithmetic operations with a clean and user-friendly interface.

---

## 🚀 Features

- ➕ Addition  
- ➖ Subtraction  
- ✖️ Multiplication  
- ➗ Division  
- 🧹 Clear / Reset functionality  
- 📱 Responsive design (works on mobile & desktop)  

---

## 🛠️ Technologies Used

- HTML5  
- CSS3  
- JavaScript (Vanilla JS)
